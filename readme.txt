Erklärung:
- Das Programm erhält 2 Argumente
  1. Inputfile Path (relativ)
  2. Outputfile Path (relativ)
- Über die Console die .exe starten unter ConsoleApp1\bin\Debug\net8.0\ConsoleApp1.exe (mit den Argumenten)
  
Compile:
- Das Projekt kann in Visual Studio compiled werden
- Mit Hilfe dieses Befehls, dotnet build ConsoleApp1\ConsoleApp1.fsproj
  -> Path der .exe, ConsoleApp1\bin\Debug\net8.0\ConsoleApp1.exe

Mit Batch File:
- Doppelclick auf RUN.bat
- Console Fenster geht auf
- war_and_peace.txt wird eingelesen vom Projektordern
- Ergebnis wird in output.txt geschrieben
